# Skill名: portfolio-site-generator
（プロフェッショナルなデザイナー向けポートフォリオサイトを制作するためのスキル）

## 🎨 デザインシステム（雪舟さんのブランド設定）
Manusは以下の色とフォントを「雪舟さんの標準」として記憶しています。

| 項目 | 設定内容 | 役割 |
|---|---|---|
| メインカラー | #C8184A (Crimson) | アクセント、ボタン、タイムライン |
| 背景色（暗） | #0F1629 (Navy) | ダークセクションの背景 |
| サブカラー | #7BAAB8 (Sky) | ダークセクションの文字アクセント |
| 背景色（明） | #FAFAF8 (White) | ライトセクションの背景 |
| 見出しフォント | Cormorant Garamond | ロゴ、大見出し、大きな数字 |
| 数字・ラベル | Space Grotesk | ナビゲーション、数字の強調 |
| 本文フォント | Noto Sans JP | 日本語の読みやすい本文 |

## 🛠 制作のワークフロー
Manusは以下の4つのステップで作業を進めるように訓練されています。

1. **情報収集**: 名前、キャッチコピー、経歴、スキル、作品、SNSリンクなどを集める。
2. **構成案の作成**: セクション構成（Hero, About, Expertise...）をMarkdownで計画する。
3. **ワイヤーフレーム**: レイアウトだけを確認するための「骨組み」を作る。
4. **高精度モックアップ**: 実際の画像や色を適用した「完成版」を作る。

## 📏 重要なデザインルール
雪舟さんのこだわりを反映した、Manusが守るべきルールです。

- **数字の強調**: セクション番号や実績の数字は Space Grotesk を使い、上端を揃えて配置する。
- **セクション順序**: Hero → About → Expertise → Philosophy → Works → Process → Contact → SNS の順で構成する。
- **レスポンシブ対応**: スマホでも崩れないよう、グリッドレイアウトを適切に使い分ける。
- **リンク設定**: noteは記事一覧ページへ、お問い合わせは指定のメールアドレスへ繋ぐ。

## 🌟 このSkillをどう使う？
新しいチャットを始めたときに、一番最初にこう伝えてください：

> 「portfolio-site-generator のSkillを読み込んで、今のサイトに新しい作品を追加して」

そうすると、Manusは上記の内容をすべて読み込み、「あ、あの赤と紺のデザインですね！フォントはCormorant Garamondを使いますね！」と、すぐに作業に取り掛かることができます。
これが、雪舟さんとManusが「阿吽（あうん）の呼吸」で仕事をするための鍵になります！
